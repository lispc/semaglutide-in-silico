# 司美格鲁肽 (Semaglutide) 药物发现 in silico 复刻 — 研究路线图

> 目标：用现代计算技术在原子层面复现 Novo Nordisk 将天然 GLP-1 改造为周给药司美格鲁肽的完整逻辑链条。
>
> 核心文献：Lau et al., *J. Med. Chem.* **2015**, 58, 7370–7380; Knudsen & Lau, *Front. Endocrinol.* **2019**, 10, 155.
>
> 最后更新：2026-05-27

---

## 一、逻辑链条总览

司美格鲁肽的设计是一个**环环相扣的四步推理**。每一步解决前一步遗留下来的问题：

```
天然 GLP-1
  │ 半衰期 ~1–2 min，DPP-4 切割 Ala8-Glu9 肽键
  │
  ▼ 决策 1: Aib8 取代 Ala8 → 空间位阻阻断 DPP-4 结合
Aib8-GLP-1
  │ DPP-4 抗性解决，但仍被肾脏快速清除（~数分钟）
  │
  ▼ 决策 2: Lys34→Arg34 → 定点酰化控制（仅在 Lys26 引入脂链）
Aib8,Arg34-GLP-1(Lys26-脂链)
  │ 脂链介导 HSA 可逆结合，半衰期延长。但：
  │   - C16 单酸亲和力不够（利拉鲁肽，~13 h）
  │   - 长单酸自聚集严重，水溶性差
  │   - 大体积脂链干扰 GLP-1R 结合
  │
  ▼ 决策 3: C16 单酸→C18 二酸 → HSA 亲和力跃升（远端羧基静电锚定）
  ▼ 决策 4: γGlu-2×OEG 柔性 Linker → 隔离脂链，恢复 GLP-1R 亲和力
司美格鲁肽
  │ 半衰期 ~165 h，一周一次给药
```

这个逻辑链中的每一步都需要计算证据支撑。我们的实验设计就是沿着这条链，逐个节点提供原子层面的验证。

| 实验 | 对应决策 | 核心问题 | 方法 |
|------|---------|---------|------|
| **A** | 决策 1 | Aib8 为什么能阻断 DPP-4？ | MD + MM-PBSA 结合自由能对比 |
| **B** | 决策 2 | 为什么必须 K34R？Lys34 脂化会怎样？ | MD 对比 K34 vs R34 受体结合 |
| **C** | 决策 3 | 为什么 C18 二酸是最优的脂链？ | HSA 结合系统性 MD + MM-GBSA |
| **D** | 决策 4 | γGlu-2×OEG 如何化解空间冲突？ | Linker 变体的 GLP-1R 结合 MD |
| **E** | 全部 | 计算管线能定量复现实验 SAR 吗？ | Lau Tables 1–4 的全 SAR MD 复现 |
| **F** | 决策 3+4 | 在完整膜受体中，脂链和 Linker 如何行为？ | 全长 GLP-1R-TMD + Gs + 膜 MD |

## 二、已有文献定位

在启动实验前，我们已阅读并理解了最相关的已有计算工作：

| 文献 | 做了什么 | 没做什么（我们的空间） |
|------|---------|---------------------|
| **Frimann et al. (2023)** | 8 个酰化 GLP-1 类似物 + GLP-1R 全受体 MD（膜环境），MM-PBSA。基于 Knudsen 2000 的老 SAR | 用的是**司美格鲁肽之前的**化合物（C8–C16, γGlu linker）。**没有 Aib8、OEG Linker、C18/C20 二酸**。不涉及 DPP-4，不涉及 HSA |
| **Liu et al. (2025)** | 司美格鲁肽 + HSA 的 1.5 μs cMD + Pep-GaMD，MM-GBSA 确定 FA3 为最优位点 | **只做了一个分子**（司美格鲁肽成品）。没有做不同脂链长度的系统对比。没有做 GLP-1R 端 |
| **Frimann 博士论文 (2024)** | 论文 1 的完整版 + 多肽自聚集实验 + 熵计算方法论 | 同论文 1 |
| **Sønderby 博士论文 (2016)** | HSA 自互作 SAXS + 利拉鲁肽-HSA 结合 + RosettaDock | 实验为主，计算为辅。用的利拉鲁肽而非司美格鲁肽 |

**定位**：已有工作各自覆盖了逻辑链的片段，但没有一个统一的、从头到尾的计算叙事。我们的贡献是把四个决策放在同一个框架下，用一致的方法学串联起来。

## 三、实验方案

### 实验 A：Aib8 对 DPP-4 的结合阻断（纯 MD，无 QM）

**对应决策**：决策 1（Ala8→Aib8 阻断 DPP-4 酶解）

**核心问题**：Aib8 的双甲基是否通过空间位阻显著降低 GLP-1 N 端与 DPP-4 活性位点的结合亲和力？

**为什么必须做**：这是逻辑链的第一环。如果 Aib8 不能阻断 DPP-4，后面所有改造（脂链、Linker）都无意义——多肽在被 HSA 保护之前就被 DPP-4 切掉了。Lau 论文 Figure 4 的体内数据（Ala8 版 PK 差，Aib8 版 PK 好）需要结构层面的解释。

**方案**：
1. 获取 DPP-4 晶体结构（PDB 1NU6, 2.10 Å），修复质子化状态
2. 构建两条 N 端底物肽（残基 7–15）：WT (HAEGTFTSD...) vs Aib8 (H[Aib]EGTFTSD...)
3. 对接至 DPP-4 活性位点（S1 口袋容纳 Ala8/Aib8，催化 Ser630 靠近肽键）
4. 500 ns cMD × 3 replica × 2 systems = 3 μs
5. MM-PBSA/MM-GBSA 计算结合自由能
6. 分析 Aib8 双甲基与 S1 口袋残基的空间距离分布和 clash 计数
7. 监测 Ser630–肽键羰基碳的距离分布（反应能力指标）

**预期**：
- ΔG_bind(WT) 比 ΔG_bind(Aib8) 更负 ≥ 5 kcal/mol
- Aib8 双甲基与 S1 口袋残基（Tyr547, Trp629 等）存在持续空间冲突
- Aib8 体系中 Ser630–肽键距离分布更宽、均值更大

**资源**：~3 μs, ~2 天 (RTX 3090)

### 实验 B：K34R 的酰化控制与结构补偿

**对应决策**：决策 2（Lys34→Arg34 实现定点酰化 + 意外获得额外受体互作）

**为什么必须做**：K34R 看起来是工艺问题（避免 Lys34 被非特异性脂化），但 Lau 论文 4ZGM 晶体结构揭示 Arg34 的胍基通过水分子与 GLP-1R Glu27 形成了额外静电互作——这是"意外的结构红利"。此外，Frimann 2023 模拟了 Lys34 酰化的后果（daC16L,K34, daC8,K34）——FA 链指向 ECD 外侧而非疏水 patch，导致结合力下降——这为"为什么必须选 Lys26 而非 Lys34"提供了支持。

**方案**：
1. AF3 构建两条肽 + GLP-1R ECD 复合物：
   - 体系 B1：Aib8, Lys34-GLP-1(7-37)（K34 版，无脂链）
   - 体系 B2：Aib8, Arg34-GLP-1(7-37)（R34 版，无脂链）
   - 体系 B3：Aib8, Lys34(脂化)-GLP-1(7-37)（模拟错误酰化的后果）
2. 以 4ZGM 为模板对接
3. 500 ns MD × 3 replica × 3 systems = 4.5 μs
4. 分析：
   - B1 vs B2：Arg34–H₂O–Glu27 氢键网络 occupancy
   - B3：脂化 Lys34 对 C 端 α-螺旋与 ECD 结合界面的破坏

**预期**：
- Arg34 体系在水介导氢键网络中显示更高 occupancy
- 脂化 Lys34 体系显示 ECD 界面 RMSD 显著升高，Phe28/Leu32 与 ECD 疏水 patch 的接触频率降低

**资源**：~4.5 μs, ~3 天 (RTX 3090)

### 实验 C：HSA 脂链结合的系统性分析

**对应决策**：决策 3（从 C16 单酸到 C18 二酸的范式转移）

**为什么必须做**：这是司美格鲁肽区别于利拉鲁肽的最关键创新。需要回答三层次的问题：
1. C18 二酸结合 HSA 的哪个位点？结合模式是怎样的？（Liu 2025 已回答：FA3，但我们需要独立验证并作为后续对比的基线）
2. 为什么 C18 二酸优于更短的 C12–C16 二酸？（疏水埋藏不足）
3. 为什么 C18 二酸优于更长的 C20/C22 二酸？（Lau 假说：U 型弯折 + 熵罚）
4. 为什么二酸优于单酸？（远端羧基的静电锚定）

**方案**：
1. 以 HSA 晶体结构（1E7G）为基础，准备 FA3 位点（Liu 2025 已确定为首选位点）
2. 构建 7 种脂链侧链探针：
   - C16 单酸（利拉鲁肽型）
   - C18 单酸
   - C12 二酸、C14 二酸、C16 二酸、C18 二酸（司美格鲁肽型）、C20 二酸、C22 二酸
3. 每条侧链连接到 Aib8,Arg34-GLP-1 骨架上，对接至 HSA FA3 位点
4. 300 ns MD × 3 replica × 8 systems = 7.2 μs
5. MM-GBSA 计算结合自由能 → ΔG_bind vs 脂链长度曲线
6. Per-residue 能量分解：定量远端羧基的静电贡献、碳链的范德华贡献
7. 观察 MD 轨迹中 C20/C22 二酸是否出现 U 型弯折

**预期**：
- ΔG_bind 曲线在 C18 二酸处达到最优（最负）
- 远端羧基与 R348/R485 的静电互作是二酸优于单酸的关键（每位点贡献 −3 到 −5 kcal/mol）
- C20/C22 轨迹中脂链在 FA3 通道内出现非天然弯折，RMSF 升高

**已有文献说明**：Liu 2025 已用 cMD + Pep-GaMD 确定了 FA3 位点和 ΔG(-75.33 kcal/mol)。我们**不做新的发现**——我们验证他们的结果，然后**扩展**到系统性的脂链长度对比曲线。这个曲线是逻辑链的关键环节：它用定量数据证明了"C18 二酸是最优解"。

**资源**：~7.5 μs, ~5 天 (RTX 3090)

### 实验 D：Linker 对 GLP-1R 结合的影响

**对应决策**：决策 4（γGlu-2×OEG 隔离脂链，恢复受体亲和力）

**为什么必须做**：没有合适的 Linker，C18 二酸会贴靠肽骨架，与 GLP-1R ECD 发生空间冲突。Lau 论文 Table 3 显示 Linker 从"无"到"γGlu-2×OEG"使 potency 从 269 pM (19) 降到 6.2 pM。Frimann 2023 只做了 γGlu Linker（无 OEG），没有覆盖 OEG 重复数的系统比较。

**方案**：
1. 构建司美格鲁肽骨架（Aib8,Arg34）+ GLP-1R ECD 复合物
2. 在 Lys26 处构建 5 种 Linker 变体（均连接 C18 二酸）：
   - **无 Linker**（化合物 19，EC50 = 269 pM）
   - **γGlu**（化合物 20，EC50 = 9.9 pM）
   - **γGlu-1×OEG**（化合物 21，EC50 = 4.8 pM）
   - **γGlu-2×OEG**（司美格鲁肽，EC50 = 6.2 pM）
   - **γGlu-3×OEG**（化合物 24，EC50 = 27.7 pM）
3. 500 ns MD × 3 replica × 5 linkers = 7.5 μs
4. 分析：
   - 脂链-ECD 距离分布（空间冲突指标）
   - ECD 结合界面 RMSD
   - Linker 区域的氢键网络（Frimann 2023 发现带电 Linker 与 Arg131 形成 H-bond）

**预期**：
- 无 Linker 和 γGlu-only 体系：脂链紧贴 ECD，导致配体-ECD 界面部分解离
- γGlu-2×OEG：脂链远离 ECD，界面稳定
- γGlu-3×OEG：过度柔性导致 entropic penalty，结合略有下降（与 EC50 = 27.7 pM 一致）

**已有文献说明**：Frimann 2023 做了 γGlu Linker（他们的"liraglutide"体系），但测试的是较短 FA 链（C14–C16）且无 OEG。我们没有在重复——我们在覆盖他们没做的 Lau 2015 设计空间（OEG 重复数 + C18 二酸）。

**资源**：~7.5 μs, ~5 天 (RTX 3090)

### 实验 E：全 SAR 表 in silico 复现

**对应决策**：全部（方法学验证）

**为什么必须做**：实验 A–D 覆盖了四个决策点的关键对比。但逻辑链需要最终验证：计算管线能**定量**复现 Lau 2015 Tables 1–4 的实验趋势吗？如果 R > 0.7，证明方法可靠；如果 R < 0.5，说明我们的力场/采样/分析策略需要修正。

**方案**：
1. 从 Lau 2015 Tables 1–4 中选择 15–20 个代表性类似物，覆盖：
   - Table 1：Arg34-GLP-1 骨架，C16/C18/C20 单酸和 C16/C18 二酸，不同 Linker
   - Table 2：Aib8,Arg34-GLP-1 骨架，C12–C20 二酸和单酸，OEG 变体
   - Table 3：C18 二酸 + 各种 Linker（γGlu, γGlu-OEG, γGlu-2×OEG, DγGlu-2×OEG, γGlu-3×OEG, Abu-2×OEG 等）
   - Table 4：不同酰化位点（Lys16/22/25/26/27/36/38）
   - 包含 3–4 个已有文献做过的类似物（如利拉鲁肽）作为跨研究交叉验证
2. GLP-1R ECD + 配体复合物构建（AF3 预测 + 4ZGM/7KI0 模板）
3. 300 ns MD × 3 replica × ~17 systems = ~15 μs
4. MM-GBSA 计算所有类似物的 GLP-1R 结合自由能
5. 对其中 6–8 个类似物，额外计算 HSA FA3 结合自由能 → 与 BR ratio 趋势对比
6. 计算 Spearman R（ΔG_MMGBSA vs log(IC50)）

**预期**：R ≥ 0.6–0.7（与 Frimann 2023 报告的 R = 0.6 一致或更好）。

**已有文献说明**：Frimann 2023 做了 8 个 Knudsen 2000 类似物的类似分析（R = 0.6），但没有覆盖 Lau 2015 的司美格鲁肽系列化合物。我们不刻意回避重叠——如果有重叠，那是好事，可以作为跨研究的方法交叉验证。但大部分类似物（特别是含 OEG Linker 和 C18+ 二酸的）是新的。

**资源**：~15 μs, ~12 天 (RTX 3090)

### 实验 F：GLP-1R 跨膜激活机制（全项目顶峰实验）

**对应决策**：决策 3+4（在完整膜受体环境中验证脂链和 Linker 的行为）

**为什么必须做**：实验 D 只在 ECD 层面分析了 Linker 的作用。但真正的 GLP-1R 是跨膜 GPCR——脂链不仅可能干扰 ECD，还可能插入细胞膜或与 TMD 接触。7KI0 cryo-EM 结构显示脂链有两种构象（接触 ECD 或接触 TM1）。这个实验把整个故事放在正确的生物学上下文中。

**方案**：
1. 使用 CHARMM-GUI 将 7KI0（司美格鲁肽-GLP-1R-Gs 复合物）嵌入 POPC/胆固醇双分子层
2. 构建 3 组体系：
   - **WT**：Aib8, Arg34, γGlu-2×OEG-C18 二酸（真实司美格鲁肽）
   - **No-linker**：Lys26 直接连 C18 二酸
   - **Short-linker**：仅 γGlu Linker
3. 1 μs MD × 3 replica × 3 systems = 9 μs
4. 分析：
   - TM6 胞内端位移（激活态指标）
   - Polar network 残基距离（HETx 基序等）
   - 脂链在 ECD/膜/TMD 之间的空间分布
   - Gs 蛋白结合界面的稳定性

**预期**：
- WT 体系维持激活态（TM6 外移 ~10–14 Å）
- No-linker 体系显示配体 N 端脱离 TMD、TM6 回缩
- Short-linker 体系介于两者之间

**这是全项目最昂贵的实验**。~250k atoms per system，OpenMM 约 45 ns/day，~9 μs ≈ 7 天日历。

**资源**：~9 μs, ~7 天 (RTX 3090)

### 经验教训

- **GLP-1R ECD + 肽复合物结构不可直接用于 MD**：所有已发表结构（3IOL/4ZGM/6GB1/7KI0）的 ECD 与肽链均分离 >25 Å
- **FlexPepDock 对长肽无效**：28 残基肽无法通过 refinement 模式桥接 25 Å 间隙（符合 best-practice §13）
- **antechamber smi/mol2/pdb 输入均失败**：sqm/bondtype 子程序不可用，改为手动构建 GAFF2 mol2
- **tleap 加载 mol2 时需包含氢原子**：仅提供重原子导致 tleap 不加 H → VDW 重叠 → NaN

## 六、技术规范

### 力场与软件

| 组件 | 选择 | 说明 |
|------|------|------|
| 生产引擎 | **OpenMM** 为主，GROMACS 交叉验证 | OpenMM 快 30–50%（best-practice §1） |
| 蛋白力场 | **ff14SB**（非 ff19SB） | ff19SB 的 CMAP 类型不支持非标准 Aib 残基 |
| 水模型 | TIP3P | 与力场配套 |
| 非标准残基 | AmberTools tleap + ParmEd 修改 | Aib8: WT→ALA→ParmEd→AIB; 脂化 Lys: GAFF2 手动 mol2 |
| 脂链配体力场 | GAFF2 + 标准官能团电荷 | antechamber 不可用（sqm/bondtype 子程序失败），手动构建 mol2 |
| 对接 | LightDock + Rosetta | 多方法交叉验证（best-practice §39） |
| 结构预测 | AF3 + Boltz-2 + Chai-1 | 用于构建配体-受体复合物初始模型 |

### 实验设计铁律（来自 best-practice.md）

1. **每条件 ≥3 replica**，报告 replica 间变异系数（§11）
2. **frame-0 验证**：clash check + 关键距离 + 初始能量（§35）
3. **PBC 处理**：GROMACS 轨迹先 `gmx trjconv -pbc whole`（§9）
4. **分析脚本独立验证**：Python RMSD 与 `gmx rms` 核对（§16）
5. **correlated t-test**，不可以用独立样本 t-test（§32）
6. **跨引擎验证**：每个实验类型至少选 1 个体系在两种引擎中各跑 20 ns（§7）
